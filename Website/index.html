Avatar Settings Manager Lite — VPM Package Listing

com.staples.asm-lite

Drop-in VRCFury prefab for VRChat avatars. Auto-discovers expression parameters at build time and adds Save / Load / Reset across three preset slots — no manual setup required.

https://vrc-staples.github.io/AvatarSettingsManager-Lite/index.json
Latest Release GitHub

Install via VCC

  1. Open VRChat Creator Companion.
  2. Go to Settings → Packages → Add Repository.
  3. Paste the listing URL above and click Add.
  4. Open your avatar project and add Avatar Settings Manager - Lite from the package list.
  5. In Unity, open Tools → .Staples. → ASM-Lite, select your avatar, and click Add ASM-Lite Prefab.
